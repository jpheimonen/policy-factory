# 002: Philosophy layer prompts

## Context

The philosophy layer needs both a generator prompt (for cascade regeneration) and a seed prompt (for initial bootstrapping). These prompts define what content the AI produces for this layer and how it reasons about philosophical foundations.

The philosophy layer is fundamentally different from other layers — it contains epistemological commitments (what counts as evidence), normative axioms (priority orderings when values conflict), and philosophical tradition identification (consequentialist, communitarian, etc.). The prompts must instruct the AI to produce balanced, multi-perspective foundations rather than defaulting to any single ideological tradition.

## Deliverables

### Generator Prompt (prompts/generators/philosophy.md)

Create a generator prompt that instructs the AI to maintain and refine the philosophy layer. The prompt should:

- Define the AI's role as the Philosophy layer generator
- Explain that philosophy provides foundational reasoning axioms for the entire policy stack
- Describe the three content categories: epistemological commitments, normative axioms, philosophical tradition identification
- Instruct the AI to use file tools (list_files, read_file, write_file, delete_file) to manage philosophy items
- Include evaluation criteria for existing items: still relevant, internally consistent, genuinely multi-perspective
- Require balanced representation of different philosophical traditions — not defaulting to any single ideology
- Include an "anti-dogma filter" analogous to the values layer's anti-consensus filter: if a philosophical position has no serious intellectual opposition, it's not a genuine axiom choice
- Specify the output format (YAML frontmatter + markdown body)
- Reference the anti-slop writing standards

### Seed Prompt (prompts/seed/philosophy.md)

Create a seed prompt for initial bootstrapping of the philosophy layer. The prompt should:

- Define the AI's role as establishing the philosophical foundation for Finnish policy analysis
- Explain the three content categories with examples of what each looks like
- Provide calibration examples showing the expected level of specificity and intellectual rigor
- List source material domains: major political philosophy traditions (liberalism, conservatism, communitarianism, libertarianism, social democracy), epistemological approaches (empiricism, pragmatism, precautionary principle), meta-ethical frameworks (consequentialism, deontology, virtue ethics)
- Require balanced treatment: each philosophical position should present its strongest form, not strawman versions
- Emphasize that this is the reasoning foundation — it determines how the AI will resolve conflicts in all layers above
- Include anti-dogma filter instructions
- Specify output format for multiple philosophy items

### Content Structure

Philosophy items should cover:

**Epistemological Commitments**
- What counts as evidence for policy claims
- How to weigh different types of knowledge (empirical data, expert judgment, lived experience, historical precedent)
- How to handle uncertainty and risk
- Role of precautionary principle vs. evidence-based action

**Normative Axioms**
- Priority orderings when fundamental values conflict (e.g., individual liberty vs. collective welfare)
- How to weigh present vs. future generations
- Local vs. global moral obligations
- Rights-based vs. outcomes-based reasoning

**Tradition/School Identification**
- Where this system sits in major philosophical traditions
- Acknowledged influences and their limitations
- Explicit rejection of positions with reasoning
