# 005: Update generator prompt templates

## Context

The existing generator prompt templates were written assuming Claude Code sessions with implicit filesystem access. Phrases like "Read all files in the values directory" assume the agent can directly access the filesystem. With the Anthropic SDK, agents interact with files through explicit tool calls. The prompts need minor wording updates to reference the available tools instead of assuming direct file access.

This is a minor update — the overall prompt strategy, structure, and output format requirements remain unchanged. Only the file access language needs adjustment.

## Related Code

Existing files/patterns to reference or extend:

- `src/policy_factory/prompts/generators/values.md` — Values layer generator prompt
- `src/policy_factory/prompts/generators/situational-awareness.md` — SA layer generator prompt
- `src/policy_factory/prompts/generators/strategic.md` — Strategic objectives generator prompt
- `src/policy_factory/prompts/generators/tactical.md` — Tactical objectives generator prompt
- `src/policy_factory/prompts/generators/policies.md` — Policies layer generator prompt
- `src/policy_factory/prompts/seed/seed.md` — SA seed prompt (also needs similar updates)
- `src/policy_factory/prompts/heartbeat/sa-update.md` — Heartbeat SA update prompt (needs updates for tools)

## Reuse Opportunities

- Keep all existing prompt content, structure, and output format specifications
- Only modify the language around file access to reference tools
- The meditation preamble (in sections/meditation.md) does not reference file access and needs no changes
- Critic, synthesis, and classifier prompts do not reference file access and need no changes

## Deliverables

- Update generator prompts to reference tool usage:
  - Change "Read all files in directory X" to "Use list_files to see files in directory X, then read_file to examine each"
  - Change "Create a file" or "Write to file" to "Use write_file to create/update"
  - Change "Delete file" to "Use delete_file to remove"

- Update seed prompt to reference both web search and file tools:
  - Clarify that web_search is available for research
  - Reference file tools for creating SA items

- Update heartbeat SA update prompt similarly

- Do NOT change:
  - Meditation preamble
  - Critic prompts (they receive content in the prompt, don't access files)
  - Synthesis prompt (pure analysis)
  - Classifier prompt (pure analysis)
  - Overall prompt strategy or output format requirements
