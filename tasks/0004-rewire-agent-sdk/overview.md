# Rewire Agent Framework to claude-agent-sdk

Replace the raw Anthropic Python SDK (`anthropic`) with `claude-agent-sdk==0.1.37` across the entire agent framework. The current implementation calls the Anthropic API directly, requiring an `ANTHROPIC_API_KEY` and separate API billing. The correct approach — used by the cc-runner reference project — wraps the Claude Code CLI binary, using the user's existing Claude subscription with no API key needed. This task rewires the agent session, file tools, config, prompts, server lifecycle, and all 8 caller modules to use the correct SDK.

## Sub-tasks

| Step | Title | Done | Description |
|------|-------|------|-------------|
| [001](001.md) | Swap dependency and remove Anthropic infrastructure | [ ] | Replace `anthropic` with `claude-agent-sdk==0.1.37` in pyproject.toml. Remove all Anthropic client lifecycle code from `server/deps.py` (the three functions and globals) and `server/app.py` (the init/shutdown calls in the lifespan). Remove the meditation filter module and its test file. Simplify `build_agent_prompt()` to remove meditation preamble loading. Update `agent/__init__.py` to remove deleted exports. |
| [002](002.md) | Convert file tools to MCP format | [ ] | Add tool context infrastructure (contextvars-based `_set_tool_context`/`get_tool_context`). Add an MCP server factory function using `create_sdk_mcp_server()`. Remove old Anthropic-format tool definitions, tool sets, dispatch dict, and web search tool constant. Update tool definition and tool set tests. |
| [003](003.md) | Rework agent config for SDK tool resolution | [ ] | Replace `resolve_tools()` (which returns Anthropic API format dicts) with a function that resolves the `allowed_tools` string list and MCP tool set for each role. | [004](004.md) | Rewrite AgentSession for ClaudeSDKClient | [ ] | Replace the entire session module. Adapt transient error detection to SDK error types. Preserve retry logic with exponential backoff. Remove the `client` constructor parameter, the manual agentic loop, `_execute_tool()`, and all `MeditationFilter` usage. Rewrite session tests with SDK mocking. |
| [005](005.md) | Update all 8 caller modules | [ ] | Remove `get_anthropic_client()` imports and calls from all caller modules. Remove the `client=client` parameter from `AgentSession` construction. Remove the `RuntimeError` catch for missing Anthropic client in the seed router endpoints. |
| [006](006.md) | Update caller and integration tests | [ ] | Update the four caller test files (`test_heartbeat_orchestrator.py`, `test_critic_runner.py`, `test_classifier.py`, `test_integration_seeding.py`) to remove `get_anthropic_client` patches and mock `AgentSession` directly. Add integration-level checks for dependency removal (no `anthropic` imports in source, no `ANTHROPIC_API_KEY` references) and correct package exports. Verify all tests pass. |
