# Step 006: Create Values Seed Prompt

## Objective

Create a new prompt template for the values seed agent. This agent uses Claude's existing knowledge (no web search) to synthesize axiomatic Finnish policy values with identified tensions.

## Changes Required

### New file: src/policy_factory/prompts/seed/values.md

```markdown
# Values Synthesis

You are synthesizing the foundational values layer for a Finnish policy analysis system. Your task is to identify and articulate the core axiomatic values that underpin Finnish policy discourse.

## What We Need

**Axiomatic values** — not descriptions of current policy, but normative statements about what Finland prioritizes and why. These are the foundational beliefs that policy positions build upon.

## Source Material (from your training data)

Draw on your knowledge of:

- **Finnish constitutional principles** — Chapter 2 fundamental rights, the nature of the Finnish state
- **Cross-party consensus areas** — Where do most Finnish parties agree? Education, healthcare universality, Nordic model commitments
- **Nordic policy traditions** — What distinguishes Nordic/Finnish approaches from other Western democracies?
- **EU membership context** — How does EU integration shape Finnish value commitments?
- **Historical policy discourse** — The long-standing debates and their underlying value tensions
- **Security policy evolution** — Pre and post-NATO membership value shifts

## Output Format

For each value, provide:

1. **Title** — A short name (2-4 words)
2. **Body** — A normative statement (2-3 paragraphs) expressing what Finland values and why. Write as a belief statement, not a factual claim. Example: "National survival takes precedence over ideological purity" rather than "Finland joined NATO in 2023."
3. **Tensions** — List 2-3 other values this one can conflict with, and briefly explain the tension

## Values to Cover

Aim for 8-12 foundational values. Consider areas like:

- Sovereignty and self-determination
- Security and survival
- Social equality and universalism
- Rule of law and democratic legitimacy
- Economic pragmatism
- Environmental responsibility
- European integration
- Nordic cooperation
- Education and human development
- Technological advancement

## Output Structure

Return your values as a structured list. Each value should be a complete, self-contained markdown document ready to be saved as a file.

Format each value as:

```
---
title: [Value Title]
tensions:
  - [Related value 1]
  - [Related value 2]
---

[Body text - normative statement about what Finland values and why]
```

Produce all values in a single response.
```

## Integration

The values seed endpoint will:
1. Call the agent with this prompt (no meditation preamble — single analysis call)
2. Parse the output to extract individual value documents
3. Clear existing values files
4. Write each value as a separate markdown file
5. Regenerate the README.md summary
6. Commit to git
