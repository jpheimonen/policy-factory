# Step 015: End-to-End Verification

## Objective

Verify the complete system works: application starts, basic flows function, and streaming operates correctly. This is the final validation step.

## Verification Checklist

### Application Startup

- [ ] `uv sync` installs all dependencies including `anthropic`
- [ ] `make dev` starts the backend server without errors
- [ ] `make build` builds frontend assets without errors
- [ ] Health check endpoint returns 200 OK
- [ ] No import errors in logs at startup
- [ ] Warning logged if `ANTHROPIC_API_KEY` not set (but app still starts)

### Basic API Operations

- [ ] `GET /api/seed/status` returns seed status with counts
- [ ] `POST /api/seed/values` with auth triggers values seeding (mock API for test)
- [ ] `POST /api/seed/` with auth triggers SA seeding (mock API for test)
- [ ] Layer endpoints return data after seeding

### Frontend Integration

- [ ] UI loads in browser at http://localhost:8080
- [ ] Admin panel accessible after login
- [ ] Seed status displayed correctly
- [ ] Cascade viewer shows streaming text when cascade runs

### With Live API (Manual Verification)

With `ANTHROPIC_API_KEY` set to a valid key:

- [ ] Values seeding completes and creates value files
- [ ] SA seeding completes and creates SA files
- [ ] Cascade runs through all layers
- [ ] Streaming text appears in real-time in cascade viewer
- [ ] Meditation countdown is not visible in streamed output
- [ ] Heartbeat operations work (skim, triage, update)

### Error Handling

- [ ] Missing API key shows clear error when agent operation attempted
- [ ] Rate limit errors are retried with backoff
- [ ] Authentication errors return 401 immediately
- [ ] Context overflow returns appropriate error

## Test Commands

```bash
# Install dependencies
uv sync

# Run unit tests
uv run pytest tests/test_agent_session.py tests/test_agent_tools.py -v

# Run integration tests
uv run pytest tests/test_seed_integration.py -v

# Start dev server
make dev

# In another terminal, test endpoints
curl http://localhost:8080/api/health
curl http://localhost:8080/api/seed/status
```

## Known Limitations

- Full E2E tests with live API require valid `ANTHROPIC_API_KEY`
- Streaming verification requires manual observation in browser
- Parallel critic tests require actual API calls or complex mocking

## Success Criteria

The task is complete when:

1. All unit tests pass
2. All integration tests pass
3. Application starts without errors
4. Seeding endpoints work (with mocked or real API)
5. Cascade viewer shows streaming text
6. No references to `claude_agent_sdk` remain in codebase
