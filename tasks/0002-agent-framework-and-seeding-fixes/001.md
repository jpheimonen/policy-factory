# Step 001: Add Anthropic SDK Dependency

## Objective

Add the official `anthropic` Python SDK to the project dependencies and verify it installs correctly.

## Changes Required

### pyproject.toml

Add `anthropic` to the dependencies list:

```toml
dependencies = [
    "python-dotenv>=1.0.0",
    "fastapi>=0.128.0",
    "uvicorn[standard]>=0.40.0",
    "bcrypt>=4.0.0",
    "pyjwt>=2.8.0",
    "pyyaml>=6.0.0",
    "apscheduler>=3.10.0,<4.0",
    "anthropic>=0.40.0",  # Add this line
]
```

## Verification

- [ ] `uv sync` completes without errors
- [ ] `uv run python -c "import anthropic; print(anthropic.__version__)"` prints the version
- [ ] `uv run python -c "from anthropic import AsyncAnthropic"` imports successfully

## Notes

- The `anthropic` package provides both sync (`Anthropic`) and async (`AsyncAnthropic`) clients
- We will use `AsyncAnthropic` throughout since the FastAPI app is async
- The package includes type stubs for IDE support
