# Step 005: Update Prompt Templates

## Objective

Adjust generator prompt templates to reference tool usage instead of assuming direct filesystem access. The prompts currently say things like "Read all files in values/" — with the Anthropic SDK, agents interact through tools.

## Changes Required

### src/policy_factory/prompts/generators/values.md

Change file access language:

**Before:**
```markdown
Read all markdown files in the values directory...
```

**After:**
```markdown
Use the list_files tool to see existing values, then read_file to examine each one...
```

### src/policy_factory/prompts/generators/situational-awareness.md

**Before:**
```markdown
Read the current SA files and update them...
```

**After:**
```markdown
Use list_files to see current SA items, read_file to examine them, and write_file/delete_file to update...
```

### src/policy_factory/prompts/generators/strategic.md

Similar changes to reference tools.

### src/policy_factory/prompts/generators/tactical.md

Similar changes to reference tools.

### src/policy_factory/prompts/generators/policies.md

Similar changes to reference tools.

### src/policy_factory/prompts/seed/seed.md

Update to reference web_search and file tools:

**Before:**
```markdown
Research current Finnish situational awareness topics...
Create markdown files for each topic...
```

**After:**
```markdown
Use web_search to research current Finnish situational awareness topics...
Use write_file to create markdown files for each topic...
```

### src/policy_factory/prompts/heartbeat/sa-update.md

Update to reference both web_search and file tools.

## Notes

- **Do NOT change prompt structure or strategy** — only update the file access language
- **Meditation preamble unchanged** — it doesn't reference file access
- **Critic prompts unchanged** — they receive content in the prompt, don't access files
- **Synthesis/classifier prompts unchanged** — no file access language
