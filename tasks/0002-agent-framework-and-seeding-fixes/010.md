# Step 010: Create Shared Anthropic Client

## Objective

Initialize a shared `AsyncAnthropic` client during FastAPI lifespan startup. This provides connection pooling for concurrent agent runs (especially the 6 parallel critics) and ensures the API key is validated at startup.

## Changes Required

### src/policy_factory/server/app.py (or main.py)

Add client initialization to the FastAPI lifespan:

```python
from contextlib import asynccontextmanager
from anthropic import AsyncAnthropic
import os
import logging

logger = logging.getLogger(__name__)

# Module-level client reference
_anthropic_client: AsyncAnthropic | None = None


def get_anthropic_client() -> AsyncAnthropic:
    """Get the shared Anthropic client."""
    if _anthropic_client is None:
        raise RuntimeError("Anthropic client not initialized")
    return _anthropic_client


@asynccontextmanager
async def lifespan(app: FastAPI):
    """Application lifespan handler."""
    global _anthropic_client

    # Validate API key is present
    api_key = os.environ.get("ANTHROPIC_API_KEY")
    if not api_key:
        logger.warning(
            "ANTHROPIC_API_KEY not set — agent operations will fail"
        )

    # Initialize Anthropic client
    _anthropic_client = AsyncAnthropic(
        api_key=api_key,
        # Connection pool settings for concurrent requests
        max_retries=0,  # We handle retries ourselves
    )

    # Initialize data directory
    initialize_data_directory(get_data_dir())

    logger.info("Application startup complete")

    yield

    # Cleanup
    if _anthropic_client:
        await _anthropic_client.close()

    logger.info("Application shutdown complete")


app = FastAPI(lifespan=lifespan)
```

### src/policy_factory/server/dependencies.py

Add dependency for accessing the client:

```python
from policy_factory.server.app import get_anthropic_client as _get_client


def get_anthropic_client():
    """FastAPI dependency for Anthropic client."""
    return _get_client()
```

### Update AgentSession constructor calls

All places that create `AgentSession` instances need to pass the client:

```python
# Before
session = AgentSession(
    config=config,
    event_emitter=event_emitter,
    context_id=context_id,
    agent_label=label,
)

# After
session = AgentSession(
    config=config,
    event_emitter=event_emitter,
    context_id=context_id,
    agent_label=label,
    client=get_anthropic_client(),  # Add this
    data_dir=get_data_dir(),         # Add this
)
```

Update in:
- `cascade/orchestrator.py` — `_default_generation_runner()`
- `cascade/critic_runner.py` — `run_critics()`
- `cascade/synthesis_runner.py` — synthesis runner
- `heartbeat/orchestrator.py` — all heartbeat tier agents
- `server/routers/seed.py` — seed endpoints
- `ideas/evaluator.py` — idea evaluation agents

## Verification

- [ ] `AsyncAnthropic` client is created during FastAPI startup
- [ ] Missing `ANTHROPIC_API_KEY` logs a warning but doesn't crash startup
- [ ] `get_anthropic_client()` returns the shared client
- [ ] `get_anthropic_client()` raises RuntimeError if called before startup
- [ ] Client is properly closed during shutdown
- [ ] All AgentSession instantiations pass the client
- [ ] Concurrent agent runs (parallel critics) share the connection pool
