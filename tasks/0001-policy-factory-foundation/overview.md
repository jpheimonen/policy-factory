# Policy Factory — Foundation

## Summary

Policy Factory is an AI-powered policy analysis engine for Finland's cross-party tech policy group. It builds policy from the ground up through a five-layer model (Values → Situational Awareness → Strategic Objectives → Tactical Objectives → Policies), with each layer challenged by 6 ideological critic perspectives. The system features bidirectional feedback memos, an idea evaluation pipeline, a tiered heartbeat for automated monitoring, and a polished dual-theme UI.

This task establishes the complete application from scratch: project structure, authentication, the markdown+git data layer, SQLite storage, the agent framework with bias meditation, cascade orchestration, critic system, idea pipeline, tiered heartbeat, input classification, real-time WebSocket streaming, and a full React frontend with all pages — stack overview, layer detail, item editing, idea inbox, live cascade viewer, activity feed, admin panel, and version history.

The reference implementation for technical patterns is the ModernPath Software Factory (cc-runner) at `/home/jp/projects/modernpath/cc-runner`.
